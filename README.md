# next-react-ts

React SPA on NEXT.js framework that can send and receive rest api data from "node-graphql-mysql" repo

### Technologies Used

1. React, Next, Redux, Redux SAGA
2. SCSS
3. GraphQL Client
4. Typescript
5. Cypress
